Vanilla JS Weather App

For my Weather App I used the OpenWeatherMap API integration, to displays real-time weather data of various cities in the world, where I also created a 6-days forecast. It is possible to change the location, by entering the city name in the search bar.

It was the final project for the SheCodes Plus Workshop: Front End Developer.

Tools: InVision Freehand, HTML5, CSS3, JavaScript ES6, Bootstrap, Netlify, Github, an API and Visual Studio Code
